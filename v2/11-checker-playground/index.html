Model Checker Playground

v2 specs carry their own input domains, so the checker explores them with zero configuration — in the browser. The spec below is a bank account with deposits, withdrawals, and an overdraft guard.

  • Liveness: can the balance reach exactly 30?
  • Safety: the balance must never go negative.

Uncheck the guard and re-run: the checker finds the overdraft traces a human reviewer might not.
